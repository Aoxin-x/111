sequenceDiagram participant U as 用户 participant S as 系统 participant DB as 数据库 U->>S: 登录系统 S->>DB: 验证用户身份 DB-->>S: 验证结果 alt 登录失败 S-->>U: 显示错误信息 else 登录成功 S-->>U: 登录成功 U->>S: 查询图书 S->>DB: 查找图书信息 DB-->>S: 返回图书信息 alt 图书不存在 S-->>U: 显示图书不存在 else 图书存在 S-->>U: 显示图书信息 DB->>DB: 检查图书库存 DB-->>S: 库存状态 alt 图书已借出 S-->>U: 显示图书已借出 U->>S: 是否预约 alt 用户选择预约 S->>DB: 添加预约记录 DB-->>S: 预约成功 S-->>U: 预约成功通知 else 用户取消 S-->>U: 操作取消 end else 图书可借 DB->>DB: 检查用户借书数量 DB-->>S: 借书数量状态 alt 借书数量超限 S-->>U: 显示借书数量超限 else 借书数量未超限 S->>DB: 更新借阅记录 DB->>DB: 添加借阅记录 DB->>DB: 减少图书库存 DB-->>S: 更新成功 S-->>U: 显示借书成功 S->>S: 发送借书确认通知 end end end end
