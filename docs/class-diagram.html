类图说明

User（用户）

基类，包含学生和教师的共同属性和方法

Student（学生）

继承自User，包含学生特有的属性

Teacher（教师）

继承自User，包含教师特有的属性

Book（图书）

图书信息管理，包含库存和位置信息

BorrowRecord（借阅记录）

记录用户借阅图书的详细信息

Reservation（预约记录）

记录用户预约图书的信息

FineRecord（罚款记录）

记录用户逾期罚款信息

Librarian（图书管理员）

管理图书和用户信息

LibrarySystem（图书馆系统）

单例模式，系统核心管理类
