状态图说明

图书状态图：描述图书从入库到报废的完整生命周期，包括在馆、可借、已借出、逾期未还、挂失等状态及其转换关系。

借阅记录状态图：描述一次借阅从创建到归档的过程，包括借阅中、正常归还、逾期中、已挂失等状态。

预约状态图：描述用户预约图书的状态变化，包括预约成功、待取书、已取书、已过期、已取消等状态。

导出图片步骤：

  2. 选择"另存为图片"或"复制图像"
  3. 如需截取整个页面，可使用 Ctrl+P 打印为PDF
